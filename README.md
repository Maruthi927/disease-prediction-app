# Multiple Disease Prediction System Using Machine Learning

A comprehensive web application for predicting multiple diseases using advanced machine learning algorithms.

## 🎯 Features

- **3 Disease Predictions**: Diabetes, Heart Disease, Cancer
- **5 ML Models**: Logistic Regression, Random Forest, Gradient Boosting, SVM, Decision Tree
- **Web Interface**: Interactive Flask-based application
- **Data Processing**: Preprocessing, scaling, encoding
- **Model Evaluation**: Accuracy, Precision, Recall, F1-Score, ROC-AUC
